graph-avatar v1.0.0

Deterministic SVG avatars via recursive planar graph subdivision.
Each avatar encodes four parameters — hue, variance, density, border width — as coloured geometry.

zero dependencies TypeScript ESM + CJS MIT License

Parameters
h hue — base color 0.028
v variance — lightness spread 0.400
d density — subdivision steps 0.600
w width — border thickness 0.300
